# Java Calculator

A simple console-based calculator built using Java.  
It performs basic arithmetic operations and maintains a history of last 5 calculations.

## Features
- Addition, Subtraction, Multiplication, Division
- Handles divide by zero
- Stores last 5 operations
- Menu-driven interface

## Technologies Used
- Java
- ArrayList
- OOP Concepts

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Compile:
   javac Calculator.java
4. Run:
   java Calculator

## Example Output
Simple Calculator:
1. Add
2. Subtract
3. Multiply
4. Divide
5. History

Result: 8
